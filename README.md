# Installation

The Installation Quite Easy Just Follow Command On Your Linux Box

       $ git clone https://github.com/KaizIqbal/Straingth_conky_v0.2.git
       $ cd Straingth_conky_v0.2/
       $ sudo chmod +x install.sh
       $ ./install.sh
